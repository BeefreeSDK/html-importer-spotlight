# 🐝 BeeFree HTML Email Importer Demo Guide

This document provides step-by-step instructions for demonstrating the BeeFree HTML Email Importer during your May 27th presentation.

## 🚀 Pre-Demo Setup

1. **Before the presentation**:
   - Ensure your `.env` file is properly configured with valid API keys
   - Run `./start-demo.sh` to verify everything works
   - Have sample-email.html ready to showcase
   - Close other applications to free up system resources

2. **Technical requirements**:
   - Stable internet connection (the demo relies on BeeFree API)
   - Browser with developer tools accessible (Chrome recommended)
   - Terminal window ready for running the demo

## ⚠️ Troubleshooting During Demo

**If API authentication fails**: Check your .env file for proper credentials
